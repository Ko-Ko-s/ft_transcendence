

## Installation

```bash
$ npm install
```

## Installation part 2

```bash
$ npm install -g @nestjs/cli
$ npm install @nestjs/common
$ npm install -g @vue/cli
$ npm i @vue/cli-service
$ npm install --save @nestjs/sequelize sequelize sequelize-typescript
$ npm install --save pg pg-hstore
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
